# Boilerplate for React application

## Get started

-  `yarn`
- `yarn serve`

## Including

### Webpack configuration

- Webpack dev server
- Babel stage 0
- JSX support
- Sass/Style Loader

### React

- Hod Module Reloading
- Redux
- Redux Sagas
- React Router

### Misc

- Styled Components
- Babel Polyfills
